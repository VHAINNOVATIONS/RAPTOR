INTRODUCTION
------------
The Simple Rules Engine Demo module enables programmers to see how the Simple Rules Engine can be implemented in a Drupal solution.

REQUIREMENTS
------------
This module only requires that the Simple Rules Engine Core and UI modules have already been installed.  It also has the following
basic requirements.

 * Drupal 7.x
 * MySQL 5.x
 * PHP 5.x

INSTALLATION
------------
The suite appears in the admin panel under the "Decision Support" heading.

CONFIGURATION
-------------
There are no files to configure.  All access is programatic from other modules.
